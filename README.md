# smsXpress

## A tool that aims to send simultaneous SMS messages to several users.

[DEMO](http://smsxpress.herokuapp.com)

For this project, I used the following tools:

* [Twilio](https://www.twilio.com/)
* MongoDB (MongoLab)
* Nodejs
* Express
* AngularJs
